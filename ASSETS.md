# Asset policy

The MIT license covers the source code and repository documentation. It does
not grant rights to third-party characters, trademarks, generated media, or
assets supplied by users.

This public snapshot intentionally excludes:

- character reference images, sprite sheets, action packs, and ZIP bundles;
- bundled background artwork whose redistribution rights were unclear or more
  restrictive than the code license;
- brand artwork and social-preview images;
- private or generated objects stored in D1/R2.

To run the complete visual workflow, provide assets you created or are licensed
to use and redistribute. Keep private assets outside Git. Runtime-generated
media belongs in owner-isolated storage and must not be committed automatically.

Wikipedia, Wikidata, and Moegirl content is fetched only after a user opts into
research. Preserve source attribution and the license metadata returned by the
research API. In particular, content marked noncommercial must not be reused in
a commercial export.
